# ClaimVault Web

Next.js 15 App Router project that explains the product story and surfaces
basic API health information. The layout is intentionally minimal so we can
introduce real dashboards once the backend collects live proofs.

## Scripts
- `pnpm dev` – run Next.js in development mode
- `pnpm build` – compile the production bundle
- `pnpm lint` – run ESLint and Next.js checks
